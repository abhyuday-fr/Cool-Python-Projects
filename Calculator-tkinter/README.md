# 🧮 Tkinter Calculator

A simple yet elegant **Calculator App** built using **Python** and **Tkinter** GUI library.  
This calculator performs basic arithmetic operations like addition, subtraction, multiplication, and division — all inside a clean, interactive graphical interface.

---

## 🚀 Features

- ➕ **Basic Operations:** Add, subtract, multiply, and divide  
- 🧑‍💻 **User-Friendly Interface:** Simple layout with clickable buttons  
- 🪟 **Responsive Design:** Buttons and display scale neatly  
- 🔢 **Keyboard Support (Optional):** Perform calculations using your keyboard  
- ⚡ **Error Handling:** Handles invalid inputs gracefully  

---

## 🧰 Technologies Used

- **Python 3.x**
- **Tkinter** – for building the graphical user interface

---

## ⚙️ Installation & Usage

1. **Clone this repository**
   ```bash
   git clone https://github.com/<your-username>/tkinter-calculator.git
   cd tkinter-calculator
   ```
   
2. Run the script
   ```bash
   python calculator.py
   ```
---

## 💡 How It Works

1. The app uses Tkinter’s Button, Entry, and Label widgets to create the calculator layout.

2. Each button is linked to a callback function that updates the display or performs calculations.

3. The eval() function (or a custom parser) computes expressions dynamically.

4. The result is displayed in the entry widget in real-time.
